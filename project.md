# Rozy Studio — Личный цифровой проект

## Что это за проект

Rozy Studio — личное цифровое пространство Разили Сунгатовой, нейростратега и AI-специалиста.
Это место, где собираются все проекты: визитки, лендинги, AI-продукты и блог.
Проект в активном развитии — сейчас это начало пути, каждый новый файл это шаг в новом направлении деятельности.

## Для кого

Для личного бренда Разили: чтобы показывать себя как эксперта, представлять продукты и делиться опытом с аудиторией.

## Стиль дизайна

**Тема:** тёмная, почти чёрная (`#080810`)

**Акценты:** фиолетовый `#a855f7` + индиго `#6366f1`, градиентные переходы между ними

**Типографика:** системный шрифт (`-apple-system`, `Segoe UI`), жирные заголовки с отрицательным трекингом, градиентный текст на ключевых словах

**Фон:** размытые цветные орбы + точечная сетка создают глубину

**Карточки:** тёмный фон `#0e0e1a`, тонкая фиолетовая рамка, скруглённые углы (18–28px), hover-эффекты с подсветкой

**Анимации:** плавное появление (`fadeUp`), пульсирующие кнопки CTA, мягкие hover-переходы

**Адаптивность:** mobile-first, `clamp()` для шрифтов, CSS Grid с `auto-fit`
