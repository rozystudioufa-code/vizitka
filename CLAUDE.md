# Правила работы с проектом Rozy Studio

## Структура файлов
- Каждая страница — один самодостаточный HTML-файл
- Никаких внешних библиотек, фреймворков и CDN
- SVG-иконки встраиваем прямо в HTML
- CSS и JS пишем внутри того же файла

## Дизайн
- **Фон:** тёмный (близко к чёрному)
- **Текст:** светлый
- **Акцентные цвета:** фиолетовый `#a855f7` + индиго `#6366f1`, градиентные переходы между ними
- Карточки с тёмным фоном, тонкой рамкой и скруглёнными углами
- Плавные анимации появления, hover-эффекты на кнопках и карточках
- Фоновые орбы + точечная сетка для глубины

## Адаптивность
- Mobile-first: сначала думаем про телефон, потом про десктоп
- Шрифты через `clamp()` — масштабируются плавно
- Сетки через CSS Grid с `auto-fit` и `minmax()`
- Не использовать `overflow-x: hidden` на `body` — ломает iOS Safari

## Стиль текста
- Простой и дружеский язык, без канцелярита
- Говорим с читателем на «ты»
- Короткие предложения, без воды

## Перед созданием нового файла
Сначала рассказать что планируется сделать: структура страницы, секции, логика — и только после согласования приступать к коду.
